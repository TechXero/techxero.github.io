For EFLinux

About

This is where EFLinux Grub Theme dedicated to his fans will be. Constantly updated so keep coming back. I included a script that makes installing much easier, just follow steps below :

1- Git Clone Repo

git clone https://github.com/TechXero/EFLinuxGrub.git

2- Set Exec permissions

sudo chmod +x ./install.sh

3- Install Theme

sudo ./install.sh

Enjoy ;)
