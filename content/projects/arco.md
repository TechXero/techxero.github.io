### SH Scripts Note

Do not forget to set all *.sh* scripts to executable prior to running them via command below..

#### Command

```bash
sudo chmod +x scriptname.sh
```

After which you can run script with this command..

#### Command

```bash
sudo ./scriptname.sh
```
